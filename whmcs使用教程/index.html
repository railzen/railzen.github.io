# WHMCS VM Push and Refund 教學

# Push

# 先備資料

  • email
  • VM ID

# WHMCS 端操作

進到 WHMCS 管理產品的介面 ( https://billing.yourdomain.com/admin/clientsservices.php?id=服務編號 (opens new window) )，找到右邊有個 More，點下去會看到 Transfer Ownership。

  • 記得先確認產品和使用者是不是正確的。

按下去之後，他會跳出新視窗，就可以使用電子郵件來找到要轉移的帳號，或是你知道 User ID 的話也可以用 User ID 直接轉移。

# VirFusion 端操作

進到 VM 管理介面 ( https://vm.yourdomain.com/admin/servers/VMID (opens new window) )，找到 change owner。

用 email 找到要轉移的使用者，按下 Transfer，完成。

# refund

# 先備資料

  • email
  • VM ID
  • WHMCS 點點點就好，不用進 VF。。。

# WHMCS 端操作

進到產品管理頁面，找到 Terminate 按下去，VirFusion 的 VM 就會消失了。

記得調整 Status 成 Completed，按儲存，之後操作 Invoice 退款即可。
